# Open Telemetry 

OpenTelemetry는 telemetry(trace, metric and logs) 데이터를 만들고 관리하는 API, SDK, 도구 통합 세트입니다.  

- 오픈 텔레메트리 ( OpenTelemetry ) = 오픈 트레이싱(Open Tracing) + 오픈 컨센서스 (Open Concensus)

## 옵저버빌리티의 핵심 컴포넌트 

- 메트릭 ( Metrics )
- 로그 ( Logs )
- 트레이싱 ( Traces )
